Esta carpeta contiene insumos locales entregados por el area usuaria para construir el PRD, las specs y los planes de implementacion de `SED_ESAL`.

Archivos locales esperados:

- `BASE DE DATOS - REGISTRO_1.xlsx`
- `Base excel.xlsx`
- `REGLAS (1).xlsx`
- `Maqueta Buscador.xlsx`
- `Plantilla Certificado EYRL.docx`
- `Prototipo/`

Por cautela, los archivos fuente no se versionan inicialmente en GitHub. Si la DIV/SED autoriza su publicacion en el repositorio, se debe ajustar `.gitignore` y registrar la decision en `docs/CONSTITUTION.md` o en la spec afectada.
